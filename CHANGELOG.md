# Changelog

All notable changes to Office Ninja Pro.

## [3.2] - 2024-12-09

### Added
- **Tab Disguise Feature** - Instantly change tab title and favicon
  - 4 presets: Google Sheets, Gmail, Google Docs, Google Calendar
  - One-click disguise from popup
  - Restore original button
- **Decoy Tabs** - Auto-open work tabs on panic
  - Configurable in Options page
  - Choose from: Docs, Sheets, Gmail, Calendar, Drive
  - Triggered by Boss Key or Panic button
- New disguise UI section in popup
- Decoy tabs settings in Options page

### Changed
- Complete code refactor with natural language comments
- Improved variable naming for readability
- Updated manifest to version 3.2
- Enhanced error handling throughout

---

## [3.0] - 2024-12-09

### Added
- **Premium UI Redesign** - Dark theme with glassmorphism
- **Quick Presets** - Night, Focus, Office Stealth modes
- **Color Themes** - Black, Midnight Blue, Forest Green, Sepia
- **Per-Site Settings** - Custom configs per website
- **Floating Widget** - Draggable controls on page
- **Boss Key** - Emergency escape shortcut (Alt+Shift+B)
- **Usage Statistics** - Track stealth time
- **Options Page** - Advanced settings management
- **Chrome Sync** - Settings sync across devices

### Changed
- New folder structure for better organization
- Separated CSS from HTML files
- Added utils/storage.js for centralized storage

### Fixed
- Widget duplication issue
- Overlay persistence on dynamic pages
- Async message handling errors
